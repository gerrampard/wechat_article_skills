# QR Codes Directory

This directory contains QR code images for the project README.

## Required Images

Please add the following QR code images to this directory:

### 1. Personal WeChat QR Code
- **Filename**: `personal_wechat.png`
- **Purpose**: For users to add your personal WeChat
- **Recommended size**: 430x430 pixels or larger
- **Format**: PNG, JPG, or JPEG

### 2. Group WeChat QR Code
- **Filename**: `group_wechat.png`
- **Purpose**: For users to join the WeChat group chat
- **Recommended size**: 430x430 pixels or larger
- **Format**: PNG, JPG, or JPEG

### 3. Appreciation QR Code
- **Filename**: `appreciation.png`
- **Purpose**: For users to support your project
- **Recommended size**: 430x430 pixels or larger
- **Format**: PNG, JPG, or JPEG

### 4. Official Account QR Code
- **Filename**: `公众号.png`
- **Purpose**: For users to follow your WeChat Official Account
- **Recommended size**: 430x430 pixels or larger
- **Format**: PNG, JPG, or JPEG

## How to Add Images

1. Save your QR code images with the exact filenames listed above
2. Place them in this `qrcodes/` directory
3. The README.md will automatically display them

## Example Structure

```
qrcodes/
├── README.md              # This file
├── personal_wechat.png    # Your personal WeChat QR code
├── group_wechat.png       # Group chat QR code
├── appreciation.png       # Appreciation/donation QR code
└── 公众号.png             # Official Account QR code
```

## Tips

- Use high-quality images for better scanning experience
- Test scanning with WeChat before publishing
- Update images regularly if they expire (especially group QR codes)
